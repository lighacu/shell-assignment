# shell scripting assignment 

## objective
this project demonstrates EC2 monitoring using bash scripting and git.

## files included
- monitor.sh
- monitor_logs.txt

## features
- shows running services
- shows CPU usage
- shows memory usage
- shows disk usage
- logs monitoring output into monitor_logs.txt
- runs automatically using cron

## cron schedule
the script is scheduled to run daily using crontab.

## github
all project files are pushed to github repository
